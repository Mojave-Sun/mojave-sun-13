code/datums/elements/connect_loc.dm

/datum/element/connect_locThis element hooks a signal onto the loc the current object is on. When the object moves, it will unhook the signal and rehook it to the new object.
/datum/element/connect_loc_behalfThis element behaves the same as connect_loc, hooking into a signal on a tracked object's turf It has the ability to react to that signal on behalf of a seperate listener however This has great use, primarially for components, but it carries with it some overhead So we do it seperately rather then intigrating the behavior with the main element
